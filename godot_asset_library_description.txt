Addon for integrating your Godot-based simulator or application with the RTI (Runtime Infrastructure) of Inhumate Suite. Use it for connecting web-based interfaces, co-simulation, user testing etc.

Read more at https://inhumatesystems.com/products/suite/.

Installation instructions:
1. Add the asset to your project
2. Project > Tools > C# > Create C# solution
3. Add the Inhumate.RTI package to the C# project: dotnet add package Inhumate.RTI
4. Build project (hammer icon)
5. Project Settings > Plugins tab > check Enabled

More documentation at https://docs.inhumatesystems.com/integrations/godot/.
